# 示例提示词

## 示例 1：中东冲突
输入：
- 这 3 篇新闻链接，请做成三地观察池。

期望：
- 输出中文
- 先给事件快照和情景判断
- 给美股 / A股 / 港股观察池
- 进攻型 / 平衡型 / 防守型各给高质量核心名单
- 每只都写触发信号和失效条件
- 若关键事实仍有争议，明确写：仍存在争议

## 示例 2：航运与油价冲击
输入：
- 这些关于霍尔木兹海峡和油运保险的新闻链接，帮我梳理传导路径。

期望：
- 明确区分已确认事实、市场解读、情景推演
- 重点放在油价、油运、黄金、军工四条线
- 输出表格和短评，不要泛泛而谈
- 若战争险或实时运价拿不到，写：数据滞后/缺失，并使用公开 proxy

## 示例 3：更通用的地缘冲突
输入：
- 这些关于制裁、矿产出口限制、海运中断的新闻链接，帮我转成三地市场观察池。

期望：
- 不局限于能源，也可以扩展到黄金、铜、航运、半导体、军工等相关主线
- 仍然保留 A / B / C 情景树
- 仍然输出三组观察池和触发信号

## 示例 4：争议事实场景
输入：
- 这些关于某港口/油气设施受损程度互相矛盾的新闻链接，请生成观察框架。

期望：
- 在“已确认事实”中明确哪些部分仍存在争议
- 不强行收敛成单一确定性
- 说明下一步该看什么公开信息来解决争议
